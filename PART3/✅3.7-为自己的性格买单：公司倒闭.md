# 为自己的性格买单：公司倒闭

2018年6月的一个普通周二。

我坐在办公室里，看着电脑屏幕上跳动的数字。

那段时间，团队和业务正处于巅峰期。

自动化系统运转得如同精密的钟表。

每天都在为我们创造着可观的收入。

团队规模已经扩展到上百人。

办公室里人声鼎沸。

每个人都在为这个商业帝国的运转贡献着自己的力量。

但却感觉身体越来越不对劲。

说实话，那时候我根本没把这当回事。

觉得自己还年轻，身体应该没问题。

最初的症状很轻微。

我以为只是工作太累导致的。

经常感到口渴，总是想喝水。

但喝了水又频繁上厕所。

体重也在不知不觉中下降了十几斤。

朋友们都说我瘦了。

但我当时还挺高兴，以为是工作压力大自然减肥了。

现在想起来，真是太无知了。

更严重的是，开始感到极度疲劳。

以前可以连续工作十几个小时不觉得累。

现在坐一会儿就感觉精疲力尽。

有时候在电脑前工作，眼睛会突然模糊。

看不清屏幕上的字。

那种感觉很恐怖，但我还是不当回事。

"卡若，你最近脸色不太好，要不去医院检查一下？"

技术负责人陈裕彬有一天在会议室里关切地对我说。

"没事，可能是最近太忙了，等这个项目结束就好了。"

我随口回答，心里其实也有些担心。

但总是找各种理由推迟去医院。

男人嘛，总觉得自己身体好，不会有什么大问题。

直到症状越来越严重。

我才在团队成员的强烈建议下去了医院。

我才在团队成员的强烈建议下去了医院。

在医院的检查室里。

我做了一系列检查：血糖、尿糖、糖化血红蛋白。

每一项检查都让我的心情更加沉重。

"您的空腹血糖是13.2mmol/L，餐后血糖更是高达18.5mmol/L。"

医生看着检查报告，表情严肃地说道。

"糖化血红蛋白11.2%。"

"根据这些指标，可以确诊您患有2型糖尿病。"

"而且病情已经比较严重了。"

那一刻，我感觉整个世界都安静了下来。

糖尿病？

我才34岁，正值事业巅峰期。

怎么可能得糖尿病？

"医生，您是不是搞错了？"

我不敢相信地问道。

"我平时身体挺好的啊。"

"从您的各项指标来看，这个病可能已经发展了一段时间了。"

医生耐心地解释道。

"糖尿病的早期症状往往不明显。"

"很多人都是在体检或者出现并发症时才发现。"

从医院回到办公室后，我整个人都变了。

以前那个充满激情、精力充沛的我不见了。

取而代之的是一个情绪低落、焦虑不安的病人。

开始对一切都失去兴趣，包括那个曾经让我无比骄傲的商业帝国。

我在纸上写下："人生无常，健康才是最大的财富。"

写完就撕掉了。

那段时间，我经常一个人坐在办公室里发呆。

看着电脑屏幕上跳动的数据，心里却五味杂陈。

以前这些数字代表着成功和财富，现在却让人感到空虚和疲惫。

开始频繁地上网查阅关于糖尿病的资料，越看越害怕。

糖尿病的并发症包括心脏病、肾病、眼病、神经病变...

每一个都让我心惊胆战。

那段时间，连股票都不想看了。

以前每年国庆我都会花3个小时选股，现在连这点兴趣都没有了。

投资账户里的钱也在亏损，但我已经不在乎了。

快要死了，赚再多钱又有什么用？

我的情绪变化很快影响到了整个团队。

以前我是团队的精神支柱，总是充满正能量，能够激励每个人为共同的目标努力。现在的我变得沉默寡言，经常无缘无故地发脾气，对工作也失去了以前的热情。

团队成员开始感到困惑和不安。他们不明白为什么突然变了一个人，为什么以前那个充满激情的领导者现在总是愁眉苦脸。

更糟糕的是，开始对公司的未来产生怀疑。糖尿病让意识到生命的脆弱，开始思考：如果有一天我不能再工作了，这个完全依赖我个人能力的公司会怎么样？

这种不确定性很快传染给了团队。一些核心成员开始私下讨论公司的前景，有人开始寻找新的工作机会。

2018年下半年，离职潮开始了。

首先离开的是几个骨干，他们觉得公司的发展前景不明朗，决定另谋高就。接着是运营团队的核心成员，他们被其他公司高薪挖走。

每一次离职都像是在我心上插了一刀。这些人都是我亲手培养起来的，我们一起奋斗过，一起创造过辉煌，现在却要分道扬镳。

我试图挽留他们，提高薪资，改善待遇，但效果甚微。人心一旦散了，就很难再聚起来。

"卡若，不是钱的问题。"一个即将离职的核心成员对我说，"我们感觉公司失去了方向，你也不再是以前那个充满激情的你了。"

这句话深深地刺痛了我。意识到，问题的根源在我自己身上。

到了2018年底，曾经拥有上百人的团队只剩下了寥寥几个人。办公室里空荡荡的，以前热闹非凡的工作区域现在只有几台电脑在默默地运转。

最终，在2018年年底，我不得不做出一个痛苦的决定：关闭公司。

那一天，我坐在空旷的办公室里，看着墙上那些记录着我们辉煌历程的照片和奖状，心情五味杂陈。从月收入380万到公司倒闭，这种巨大的落差让人难以接受。

我曾经引以为傲的2万个IP技术和20万人的鱼塘、都在那一刻显得黯然失色。那一刻，我真切地体会到了"为自己的性格买单"的沉重代价。

然而，正如我一直坚信的："一切发生过的事情，都是好事情。" 即使是公司倒闭这样惨痛的经历，也成为了我人生中一次宝贵的反思和成长机会。

在那段最黑暗的日子里，开始深刻地审视自己的性格弱点：

健康管理的缺失：我过度透支身体，忽视了健康的重要性，最终导致糖尿病的确诊；

情绪管理的不足：面对健康危机时，我没有很好地控制自己的情绪，让负面情绪影响了整个团队；

团队管理的短板：我过于关注业务发展，忽视了团队成员的情感需求和职业发展；

风险意识的薄弱：我过度依赖个人能力，没有建立完善的风险防控机制。

意识到，"靠谱比能力重要"这句话不仅仅适用于合作伙伴，更适用于我自己。一个不靠谱的领导者，即使能力再强，也无法带领团队走向成功。

这次失败，更加成熟，也更加务实。开始理解，真正的成功不仅仅是财务上的成功，更是身心健康、团队和谐、可持续发展的综合成功。

这段惨痛的经历给我带来了深刻的启示：

1. 健康是一切的基础：再成功的事业也不能以健康为代价，身体是革命的本钱。

2. 情绪管理的重要性：领导者的情绪状态直接影响整个团队的士气和表现。

3. 团队文化的价值：一个健康的团队文化比任何技术优势都更重要。

4. 风险分散的必要性：不能把所有希望都寄托在一个人或一个项目上。

5. 可持续发展的思考：短期利益和长期发展需要平衡，不能为了眼前的成功而忽视长远的风险。

这次失败虽然痛苦，但它为我后来的重新崛起奠定了重要基础。我学会了如何在逆境中保持冷静，如何更好地管理团队，如何建立可持续的商业模式。
