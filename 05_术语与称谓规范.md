# 术语与称谓规范

## 统一写法

| 统一词 | 不再使用 | 说明 |
|---|---|---|
| 卡若 | 卡罗、卡洛、卡路 | 指作者及其AI体系时统一为卡若 |
| 流量池 | 流量词 | 统一商业术语 |
| 私域 | 私域流量、私域银行打法 | 首次可解释，后续按语境简写 |
| 个人IP | IP人设、个人品牌（非必要时） | 本书核心概念 |
| 超级个体 | 一人公司、个人创业者（标题可保留） | 正文首次出现时解释 |
| 存客宝 | 工具、系统（泛称） | 产品全称首次出现，后续可简写 |

## 数字口径

数字必须保留来源等级；“约、将近、这一档”不能被改成精确结果。未有来源的增长率、客户数、收入和转化率不进入公开稿。
